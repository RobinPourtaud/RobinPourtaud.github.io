## Définition de Pi :

\[latexpage\]

Pi est certainement la constante mathématique la plus connu. Enseigné à l'école primaire, elle est notamment utilisé pour calculer le périmètre d'un cercle grâce à son rayon :

Soit P le périmètre du cercle et R son rayon, alors :

$P = 2 \\times \\pi \\times R$

Autrement dit : $\\pi = \\frac{P}{2\\times R}$

Mais on peut aussi la retrouver dans de nombreuses autre formules !

- La célèbre formule d'Euler : $e^{i\\theta}+1=0$
- La formule de Riemann : $\\zeta{2} = \\frac{\\pi^2}{6}$
- La formule de Stirling : $\\lim\_{n \\to +\\infty} {n\\,!\\over \\sqrt{2 \\pi n} \\; \\left({n}/{\\rm e}\\right)^{n} } = 1$
- Le sinus intégral sur $\\mathbb{R} = 2\\int\_{0}^{\\infty} \\frac {\\sin x}{x}dx = \\pi$
