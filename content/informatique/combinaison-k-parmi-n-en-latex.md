Combien y a-t-il de possibilités de tirer 3 cartes parmi 13 ? Vous voulez certainement parler des coefficients binomiaux et vous ne savez pas comment le faire en Latex. Ci-dessous se trouvent 2 façons de rédiger des coefficients binomiaux pour vos PDF.

\[latexpage\]

## Notation "Française" :

k parmi n peut se noter de la façon suivante :

$\\quicklatex{size=28}C\_n^k $

Son code Latex en mode mathématique est tout simplement :

```
C_n^k 
```

## Notation "Anglo-Saxonne" :

Notation certainement la plus couramment utilisée pour exprimer le coefficient binomial :

$\\quicklatex{size=28}\\binom{n}{k}$

Son code Latex en mode Mathématique est :

```
\binom{n}{k}
```
