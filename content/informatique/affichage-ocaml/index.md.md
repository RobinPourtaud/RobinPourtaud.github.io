## Afficher un charactèr
Pour afficher un charactère, il faut utiliser la fonction **print_char**. 
```ocaml
print_char 'a';;
```
Ce code affiche la lettre a dans la sortie standard. 

## Afficher un entier
Pour afficher un entier, il faut utiliser la fonction **print_int**. 
```ocaml
print_int 1;;
```
Ce code affiche l'entier 1 dans la sortie standard. 

## Afficher un nombre flotant
Pour afficher un nombre flottant, il faut utiliser la fonction **print_float**
```ocaml
print_int 1.5;;
```

## Afficher un string

```ocaml
print_string "bonjour";;
```
