## Afficher un charactère
Pour afficher un charactère, il faut utiliser la fonction **print_char**. 
```ocaml
print_char 'a';;
```
Ce code affiche la lettre a dans la sortie standard. 

## Afficher un entier
```ocaml
print_int 1;;
```
Ce code affiche l'entier 1 dans la sortie standard.
