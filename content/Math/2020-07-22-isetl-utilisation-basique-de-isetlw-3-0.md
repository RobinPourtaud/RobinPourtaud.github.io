_Interactive SET Language (ISETL) est un langage de programmation basé sur SETL. Développé par Garry Levin en 1988, elle avait pour finalité&nbsp;_**l’enseignement des mathématiques discrètes à l’université**_.&nbsp;_Cet article a pour but de vous présenter comment exécuter du code via le logiciel ISETLW 3.0 sur Windows 10. 

## Nécessaire : 

Pour pouvoir suivre ce tutoriel, il est nécessaire d&rsquo;avoir accès au logiciel ISETLW 3.0 sur Windows. Pour cela, vous pouvez vous référer à mon précédent article sur son installation : 

<p class="has-text-align-center">
  <a href="https://keskec.fr/sciences/informatique/robin/2690/" class="rank-math-link">ISETL 3.0 sur Windows 10</a>
</p>

## Utilisation plus confortable : 

ISETLW 3.0, n&rsquo;est pas un logiciel très « User-Friendly ». Cependant, il est possible d&rsquo;améliorer légèrement son utilisation en 3 étapes: 

### Étape 1 : Réduire « Graph Window » 

Commencez par mettre en cascade ISETL, pour cela appuyez sur « Window » puis « Cascade ».

Ensuite réduisez la fenêtre « Graph Windows » : <figure class="wp-block-image size-large">

<img loading="lazy" width="1024" height="211" src="https://keskec.fr/wp-content/uploads/2020/07/image-10-1024x211.png" alt="" class="wp-image-3844" srcset="https://keskec.fr/wp-content/uploads/2020/07/image-10-1024x211.png 1024w, https://keskec.fr/wp-content/uploads/2020/07/image-10-300x62.png 300w, https://keskec.fr/wp-content/uploads/2020/07/image-10-768x159.png 768w, https://keskec.fr/wp-content/uploads/2020/07/image-10-1536x317.png 1536w, https://keskec.fr/wp-content/uploads/2020/07/image-10-2048x423.png 2048w, https://keskec.fr/wp-content/uploads/2020/07/image-10-1568x324.png 1568w, https://keskec.fr/wp-content/uploads/2020/07/image-10-150x31.png 150w" sizes="(max-width: 1024px) 100vw, 1024px" /> <figcaption>Réduction de graph Window</figcaption></figure> 

### Étape 2 : Ouvrir un nouveau document

Appuyez sur « File » puis « New ». 

### Étape 3 : Redimensionnez l&rsquo;affichage

Vous pouvez maintenant cliquer sur « Window » puis « Tile Horizontally » ou « Tile Vertically » selon vos préférences : <figure class="wp-block-image size-large">
