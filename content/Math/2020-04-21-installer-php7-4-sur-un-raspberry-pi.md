PHP7.4 est actuellement la dernière version de PHP. Sortie fin 2019, je vais vous montrer la méthode la plus simple pour installer cette version de php sous Raspbian.

Vous avez peut-être remarqué que votre WordPress tourne sous une ancienne version de PHP ? Vous avez donc sans doute vu ce message :<figure class="wp-block-image size-large">

<img src="https://keskec.fr/wp-content/uploads/2020/04/image-3-1024x331.png" alt="" class="wp-image-789" /> <figcaption>Problème Critique PHP WordPress</figcaption></figure> 

Prenez 10 min de votre temps pour régler ce problème. 

## Prérequis : {.has-text-align-center}

  * Vous devez avoir un ordinateur mis à jour, sous une distribution basée sur debian (raspbian pour moi). 
  * Avoir les droits de super utilisateur et donc avoir accès à la commande « sudo ».
  * Avoir de préférence Apache2.

Pour mettre à jour votre ordinateur, rien de plus simple:

<pre class="wp-block-code"><div>
  <code lang="bash" class="hljs language-bash">sudo apt update && sudo apt upgrade</code>
</div></pre>

## Installation du repository : {.has-text-align-center}

Ajoutons la clé du repository sury avec wget.

<pre class="wp-block-code" aria-describedby="shcb-language-23" data-shcb-language-name="JavaScript" data-shcb-language-slug="javascript"><div>
  <code lang="bash" class="hljs language-javascript language-bash">wget -qO - https:&lt;span class="hljs-comment">//packages.sury.org/php/apt.gpg | sudo apt-key add -&lt;/span></code>
</div>

<small class="shcb-language" id="shcb-language-23"><span class="shcb-language__label">Code language:</span> <span class="shcb-language__name">JavaScript</span> <span class="shcb-language__paren">(</span><span class="shcb-language__slug">javascript</span><span class="shcb-language__paren">)</span></small></pre>

Ajoutons le repository sury.

<pre class="wp-block-code" aria-describedby="shcb-language-24" data-shcb-language-name="PHP" data-shcb-language-slug="php"><div>
  <code lang="bash" class="hljs language-php language-bash">&lt;span class="hljs-keyword">echo&lt;/span> &lt;span class="hljs-string">"deb https://packages.sury.org/php/ $(lsb_release -sc) main"&lt;/span> | sudo tee /etc/apt/sources.&lt;span class="hljs-keyword">list&lt;/span>.d/php7.x.&lt;span class="hljs-keyword">list&lt;/span></code>
</div>

<small class="shcb-language" id="shcb-language-24"><span class="shcb-language__label">Code language:</span> <span class="shcb-language__name">PHP</span> <span class="shcb-language__paren">(</span><span class="shcb-language__slug">php</span><span class="shcb-language__paren">)</span></small></pre>

Mettez à nouveau à jour votre ordinateur.

<pre class="wp-block-code"><div>
  <code lang="bash" class="hljs language-bash">sudo apt update && sudo apt upgrade</code>
</div></pre>

## Installation de PHP7.4 {.has-text-align-center}

Si vous aviez installer une version de PHP antérieur, il est probable que la commande précédente ait déjà installé la dernière version de PHP :).

Vous pouvez vérifier votre version de php avec 

<pre class="wp-block-code"><div>
  <code lang="bash" class="hljs language-bash">php -v </code>
</div></pre>

Sinon, vous pouvez maintenant utiliser la commande suivante pour installer PHP7.4 et un module apache.

<pre class="wp-block-code" aria-describedby="shcb-language-25" data-shcb-language-name="CSS" data-shcb-language-slug="css"><div>
  <code lang="bash" class="hljs language-css language-bash">&lt;span class="hljs-selector-tag">sudo&lt;/span> &lt;span class="hljs-selector-tag">apt&lt;/span> &lt;span class="hljs-selector-tag">install&lt;/span> &lt;span class="hljs-selector-tag">php7&lt;/span>&lt;span class="hljs-selector-class">.4&lt;/span>
  &lt;span class="hljs-selector-tag">sudo&lt;/span> &lt;span class="hljs-selector-tag">apt&lt;/span> &lt;span class="hljs-selector-tag">install&lt;/span> &lt;span class="hljs-selector-tag">libapache2-mod-php7&lt;/span>&lt;span class="hljs-selector-class">.4&lt;/span></code>
</div>

<small class="shcb-language" id="shcb-language-25"><span class="shcb-language__label">Code language:</span> <span class="shcb-language__name">CSS</span> <span class="shcb-language__paren">(</span><span class="shcb-language__slug">css</span><span class="shcb-language__paren">)</span></small></pre>

(optionnel) Vous pouvez installer directement toutes les extensions:

<pre class="wp-block-code" aria-describedby="shcb-language-26" data-shcb-language-name="CSS" data-shcb-language-slug="css"><div>
  <code lang="bash" class="hljs language-css language-bash">&lt;span class="hljs-selector-tag">sudo&lt;/span> &lt;span class="hljs-selector-tag">apt&lt;/span> &lt;span class="hljs-selector-tag">install&lt;/span> &lt;span class="hljs-selector-tag">php7&lt;/span>&lt;span class="hljs-selector-class">.4-&lt;/span>*</code>
</div>

<small class="shcb-language" id="shcb-language-26"><span class="shcb-language__label">Code language:</span> <span class="shcb-language__name">CSS</span> <span class="shcb-language__paren">(</span><span class="shcb-language__slug">css</span><span class="shcb-language__paren">)</span></small></pre>

(Si vous aviez déjà PHP) Désactivez votre ancienne version de PHP pour apache2 (dans mon cas PHP7.0)

<pre class="wp-block-code" aria-describedby="shcb-language-27" data-shcb-language-name="CSS" data-shcb-language-slug="css"><div>
  <code class="hljs language-css">&lt;span class="hljs-selector-tag">sudo&lt;/span> &lt;span class="hljs-selector-tag">a2dismod&lt;/span> &lt;span class="hljs-selector-tag">php7&lt;/span>&lt;span class="hljs-selector-class">.0&lt;/span></code>
</div>

<small class="shcb-language" id="shcb-language-27"><span class="shcb-language__label">Code language:</span> <span class="shcb-language__name">CSS</span> <span class="shcb-language__paren">(</span><span class="shcb-language__slug">css</span><span class="shcb-language__paren">)</span></small></pre>

Et bien sûr, activez php7.4

<pre class="wp-block-code" aria-describedby="shcb-language-28" data-shcb-language-name="CSS" data-shcb-language-slug="css"><div>
  <code lang="bash" class="hljs language-css language-bash">&lt;span class="hljs-selector-tag">sudo&lt;/span> &lt;span class="hljs-selector-tag">a2enmod&lt;/span> &lt;span class="hljs-selector-tag">php7&lt;/span>&lt;span class="hljs-selector-class">.4&lt;/span></code>
</div>

<small class="shcb-language" id="shcb-language-28"><span class="shcb-language__label">Code language:</span> <span class="shcb-language__name">CSS</span> <span class="shcb-language__paren">(</span><span class="shcb-language__slug">css</span><span class="shcb-language__paren">)</span></small></pre>

Vous pouvez maintenant redémarrer Apache2 :).

<pre class="wp-block-code"><div>
  <code lang="bash" class="hljs language-bash">sudo service apache2 restart</code>
</div></pre>
