## Démonstration : 

Nous allons prouvé cette propriété par le théorème de récurrence simple. 

Soit $\mathbb{N} muni de son ordre naturel. 

Propriété : Soit P(n) une assertion défini $\forall n \in \mathbb{N}, tel que $P(n):\sum_{k=0}^{n}k = \frac{n\times (n+1)}{2}$

Initialisation : Pour n = 0, nous avons bien \frac{0\times 1}{2} = 0. La propriété est vrai pour le rang $n_0$.

Hérédité : Supposons la propriété vrai pour le rang n
