## Fonction trigonométrique : 

Soit x un réel quelconque (appartenant aux [domaines de définition][2]{.rank-math-link.rank-math-link.rank-math-link} des fonctions) : <figure class="wp-block-table">

| Fonction Isetl | Fonction mathématique |
| -------------- | --------------------- |
| sin(x)         | Sinus                 |
| cos(x)         | Cosinus               |
| tan(x)         | Tangente              |
| sec(x)         | Sécante               |
| csc(x)         | Cosécante             |
| cot(x)         | Cotangente            |
| asin(x)        | Arc Sinus             |
| acos(x)        | Arc Cosinus           |
| atan(x)        | Arc Tangeante         |<figcaption>Tableau récapitulatif des fonctions trigonométriques :</figcaption></figure>
