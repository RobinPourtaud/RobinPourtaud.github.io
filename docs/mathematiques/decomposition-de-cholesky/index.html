En python :

Il est possible d’effectuer une factorisation de Cholesky rapidement en Python grâce à numpy !

import numpy as np

A = np.array([[1,2], [2,7]])
B = np.linalg.cholesky(A)
print(B)

Pour en savoir plus : numpy.linalg.cholesky.
